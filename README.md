BIOS611 Final Project

**The analysis includes:**

	•	Exploratory Data Analysis (EDA)
	•	Correlation patterns across lifestyle indicators
	•	Principal Component Analysis (PCA)
	•	K-means clustering on reduced PCA space
	•	Visualization of cluster characteristics
	•	2D and 3D PCA scatter plots
	•	Feature comparison across clusters

**Key variables considered include:**

	•	Income & rent levels
	•	Population density
	•	Air quality
	•	Internet access
	•	Transportation infrastructure
	•	Green space ratio
	•	Happiness score

**How to Reproduce**

install.packages(c("tidyverse", "FactoMineR", "factoextra", "ggplot2", "plotly", "corrplot"))

**Run the Makefile to generate the report:**

make
